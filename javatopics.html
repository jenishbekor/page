Java Tutorial: Input/Output and Collections

Java Input and Output

Use Scanner for console input and System.out for output. 

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("Enter name: ");
        String name = sc.nextLine();

        System.out.print("Enter age: ");
        int age = sc.nextInt();

        System.out.println("Hello, " + name + "! You are " + age + " years old.");

        sc.close();
    }
}

String ‚Äî Text Manipulation

Strings in Java are immutable and provide many helpful methods for processing text. 

public class StringMethods {
    public static void main(String[] args) {
        String text = "  Hello Java World!  ";

        // Trim spaces
        System.out.println(text.trim());

        // Length
        System.out.println("Length: " + text.length());

        // Access character
        System.out.println("First char: " + text.charAt(0));

        // Substring
        System.out.println("Substring: " + text.substring(2, 7));

        // Case conversion
        System.out.println(text.toUpperCase());
        System.out.println(text.toLowerCase());

        // Replace
        System.out.println(text.replace("Java", "Python"));

        // Check content
        System.out.println(text.contains("Java"));
        System.out.println(text.startsWith("  H"));
        System.out.println(text.endsWith("!  "));

        // Split string
        String[] words = text.trim().split(" ");
        for (String w : words) System.out.println(w);

        // Join strings
        String joined = String.join("-", "apple", "banana", "cherry");
        System.out.println(joined);

        // Compare
        String a = "Hello";
        String b = "hello";
        System.out.println(a.equals(b));          // false
        System.out.println(a.equalsIgnoreCase(b));// true

        // Concatenate
        String greeting = a.concat(" ").concat("Java");
        System.out.println(greeting);

        // Check empty or blank
        String s = " ";
        System.out.println(s.isEmpty()); // false
        System.out.println(s.isBlank()); // true
    }
}

üìò Useful Methods

  • length() ‚Äî number of characters
  • charAt(int index) ‚Äî get character
  • substring(int start, int end)
  • toUpperCase(), toLowerCase()
  • trim(), strip()
  • replace(old, new)
  • split(regex), join(delimiter, ...)
  • equals(), equalsIgnoreCase()
  • startsWith(), endsWith(), contains()
  • isEmpty(), isBlank()
  • concat(String s)

ArrayList ‚Äî Dynamic Array

ArrayList allows dynamic resizing, duplicates, and ordered storage. 

import java.util.*;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();

        // Add elements
        list.add("Orange");
        list.add("Apple");
        list.add("Banana");
        list.add("Mango");

        // Access and modify
        System.out.println(list.get(2)); // Banana
        list.set(2, "Cherry");

        // Remove by value or index
        list.remove("Orange");
        list.remove(0);

        // Sort elements
        Collections.sort(list);
        System.out.println("Sorted: " + list);

        // Reverse order
        Collections.sort(list, Collections.reverseOrder());
        System.out.println("Reversed: " + list);

        // Search
        System.out.println("Contains Mango: " + list.contains("Mango"));

        // Convert to array
        Object[] arr = list.toArray();

        // Iterate
        for (String item : list) {
            System.out.println(item);
        }

        list.clear();
    }
}

üìò Useful Methods

  • add(E e) ‚Äî add element
  • get(int index) ‚Äî get by index
  • set(int index, E e) ‚Äî update element
  • remove(Object o) / remove(int index)
  • contains(Object o)
  • indexOf(Object o), lastIndexOf(Object o)
  • Collections.sort(list)
  • Collections.reverse(list)

HashSet ‚Äî Unique Elements

HashSet stores only unique items and is unordered.

import java.util.*;

public class HashSetExample {
    public static void main(String[] args) {
        HashSet<String> set = new HashSet<>();

        set.add("Java");
        set.add("Python");
        set.add("C++");
        set.add("Java"); // ignored

        System.out.println("Set: " + set);

        set.remove("C++");
        System.out.println("Contains Python: " + set.contains("Python"));

        // Convert to list and sort
        List<String> sorted = new ArrayList<>(set);
        Collections.sort(sorted);
        System.out.println("Sorted: " + sorted);

        for (String lang : set) {
            System.out.println(lang);
        }

        set.clear();
    }
}

  • add(E e)
  • remove(Object o)
  • contains(Object o)

ArrayDeque ‚Äî Double-Ended Queue

ArrayDeque can be used both as a stack and as a queue.

import java.util.ArrayDeque;

public class ArrayDequeExample {
    public static void main(String[] args) {
        ArrayDeque<String> dq = new ArrayDeque<>();

        // Queue usage (FIFO)
        dq.offer("A");
        dq.offer("B");
        dq.offer("C");

        System.out.println("Front: " + dq.peek());
        dq.poll(); // remove front
        System.out.println(dq);

        // Stack usage (LIFO)
        dq.push("X");
        dq.push("Y");
        dq.push("Z");

        System.out.println("Stack pop: " + dq.pop());
        System.out.println(dq);
    }
}

üìò Useful Methods

  • add(E e), offer(E e)
  • remove(), poll()
  • peek()
  • push(E e), pop()
  • clear()
  • size()

HashMap ‚Äî Key-Value Pairs

HashMap stores data as key-value pairs, allowing fast access by key. 

import java.util.*;

public class HashMapExample {
    public static void main(String[] args) {
        HashMap<String, Integer> map = new HashMap<>();

        // Add key-value pairs
        map.put("Alice", 90);
        map.put("Bob", 85);
        map.put("Charlie", 95);

        // Access
        System.out.println("Bob's score: " + map.get("Bob"));

        // Replace value
        map.replace("Alice", 92);

        // Remove
        map.remove("Bob");

        // Iterate
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey() + " => " + entry.getValue());
        }

        // Sort by value
        map.entrySet()
           .stream()
           .sorted(Map.Entry.comparingByValue())
           .forEach(e -> System.out.println(e.getKey() + ": " + e.getValue()));

        map.clear();
    }
}

üìò Useful Methods

  • put(K key, V value)
  • get(Object key)
  • containsKey(Object key), containsValue(Object v)
  • remove(Object key)
  • replace(K key, V newValue)
  • keySet(), values(), entrySet()
  • clear()

Classes and Objects ‚Äî Basics of OOP

A class is a blueprint for creating objects. An object is an instance of a class containing data (fields) and behavior (methods).

üìå Important Concepts

  • Fields ‚Äî variables inside a class
  • Constructor ‚Äî runs when creating an object
  • Methods ‚Äî actions/behaviors of the object
  • this keyword ‚Äî refers to the current object
  • implements Comparable ‚Äî allows sorting objects
  • Collections.sort() ‚Äî sorts based on compareTo()

üìò Example: Class with Comparable for Sorting

import java.util.*;

class Point implements Comparable<Point> {
    int x, y;

    // Constructor
    Point(int x, int y) {
        this.x = x;
        this.y = y;
    }

    // Method to display point
    void display() {
        System.out.println("(" + x + ", " + y + ")");
    }

    // Compare logic for sorting
    @Override
    public int compareTo(Point point) {
        if (this.x == point.x)
            return this.y - point.y;  // sort by y if x is same
        return this.x - point.x;      // sort by x
    }
}

public class Main {
    public static void main(String[] args) {
        Point p1 = new Point(2, 5);
        Point p2 = new Point(1, 3);
        Point p3 = new Point(2, 1);

        ArrayList<Point> points = new ArrayList<>();
        points.add(p1);
        points.add(p2);
        points.add(p3);
        points.add(new Point(0, 1));
        points.add(new Point(3, 0));
        points.add(new Point(3, 1));

        // Sorting points
        Collections.sort(points);

        // Displaying sorted points
        for (Point p : points) {
            p.display();
        }
    }
}

üìå Output Explanation

  • Points are sorted first by x.
  • If x is the same, they are sorted by y.
  • This is possible because the class implements Comparable<Point>.
